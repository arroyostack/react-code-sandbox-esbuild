# ESBuild-wasm Learning Repository

This repository serves as a practical guide for learning how to use ESBuild-wasm to transpile and bundle code directly in the browser. The project is a small web app where users can write code, import npm libraries, and see the results instantly in their browser.

This tutorial was taught by Stephen Grider.

### Installation

1. Clone this repository:

   ````yarn

   ```yarn dev
   ````
